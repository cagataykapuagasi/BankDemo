# BankDemo

Demo for test.

## Installation

```bash
git clone git@github.com:cagataykapuagasi/BankDemo.git
yarn or npm install
```

## Ios

```bash
cd ios && pod install
react-native run-ios
```

## Android

```bash
react-native run-android
```
